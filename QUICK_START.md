# 快速启动指南

## 🚀 5分钟快速上手

### 步骤1：配置API地址（可选）

如果你有自己的视频API，编辑以下文件：

```kotlin
// app/src/main/java/com/example/tv/api/RetrofitClient.kt
private const val BASE_URL = "https://your-api-domain.com/" // 改成你的API地址
```

**注意**：如果API未配置或失败，应用会自动使用内置的模拟视频数据，不影响使用。

### 步骤2：添加本地视频（可选）

将你的视频文件（.mp4格式）复制到：
```
app/src/main/assets/
```

当前已有两个示例视频：
- Intergalactic-Space-Room-4K.mp4
- Samurai-Revenge-4K.mp4

### 步骤3：编译运行

1. 在Android Studio中打开项目
2. 连接Android TV设备或启动TV模拟器
3. 点击运行按钮（或按 Shift+F10）

### 步骤4：设置壁纸

1. 应用启动后，点击"设置视频壁纸"按钮
2. 在系统壁纸选择界面，选择"TV"
3. 点击"设置壁纸"

### 步骤5：切换视频

**使用遥控器**：
- 按 **左键** ← 切换到上一个视频
- 按 **右键** → 切换到下一个视频

**使用应用界面**：
- 打开应用，点击"← 上一个"或"下一个 →"按钮

## 📱 视频源说明

应用会按以下顺序加载视频：

1. **本地视频**（assets文件夹）- 立即可用
2. **远程视频**（API接口）- 异步加载
3. **模拟视频**（内置）- API失败时使用

所有视频会合并到一个播放列表中，可以循环切换。

## 🔧 常见问题

### Q: 壁纸设置后看不到视频？
A: 返回主屏幕查看，壁纸在桌面背景显示。

### Q: 遥控器按键没反应？
A: 确保已正确设置壁纸，可以先用应用内按钮测试。

### Q: 视频播放卡顿？
A: 尝试使用分辨率较低的视频（建议1080p）。

### Q: 远程视频加载失败？
A: 检查网络连接，应用会自动使用模拟数据。

## 📝 API接口格式

如果要对接自己的API，返回格式应为：

```json
{
  "videos": [
    {
      "id": "video_1",
      "url": "https://example.com/video1.mp4",
      "title": "视频标题"
    }
  ]
}
```
